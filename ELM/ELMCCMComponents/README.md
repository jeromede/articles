# [CCM](https://jazz.net/downloads/workflow-management) Components in [ELM](https://jazz.net/)

## Prerequites

IBM Engineering Workflow Management installed (one or more server machines) and locally (workstation): Client for Eclipse IDE and/or Windows shell integration (if Windows).

[Streams and components](https://www.ibm.com/docs/en/engineering-lifecycle-management-suite/workflow-management/7.1.0?topic=control-streams-components) (the ones from CCM, not GC).
