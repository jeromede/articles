# [CCM](https://jazz.net/downloads/workflow-management) Components in [ELM](https://jazz.net/)

## Prerequites


Server:

* IBM Engineering Workflow Management installed (one or more server machines)

Locally (workstation), one or more of:

* Client for Eclipse IDE
* Visual Studio IDE integration
* Windows shell integration (if Windows)
* SCM (command line)

## References (version 7.2)

[Streams and components](https://www.ibm.com/docs/en/engineering-lifecycle-management-suite/workflow-management/7.2.0?topic=overview-streams-components) (the ones from CCM, not GC).

[Adding and removing components](https://www.ibm.com/docs/en/engineering-lifecycle-management-suite/workflow-management/7.2.0?topic=environments-adding-removing-components).

[Organizing components into hierarchies](https://www.ibm.com/docs/en/engineering-lifecycle-management-suite/workflow-management/7.2.0?topic=client-organizing-components-into-hierarchies).

## Scenario

We want a CCM project area with a stream that contains a hierarchy of components.
